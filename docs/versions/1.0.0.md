# All We Experience 1.0.0

작성일: 2026-04-11  
앱 버전: `1.0.0`

## 목적

`All We Experience`는 아이의 잘한 일과 아쉬운 일을 빠르게 기록하고, 날짜별 / 기간별 흐름을 로컬에서 확인하기 위한 개인용 앱이다.

## 현재 제품 범위

### 1. 메인 탭: 업다운

- 메인 탭 이름은 `업다운`
- 핵심 행동은 `엄지 척!`, `이건 좀...` 입력
- 메모 입력 후 기록 가능
- 오늘 점수와 오늘의 상승 / 하락 건수 표시
- 최근 기록 목록 표시
- 롱프레스 반복 입력은 최근 기록에서 `묶음 1건`으로 표시
- 묶음 입력은 화살표 배지 숫자로 점수 표시
- `1점` 입력은 숫자 배지를 표시하지 않음
- 최근 입력 직후 하단 `undo snackbar`로 되돌리기 가능

### 2. 달력 탭

- 현재 월 기준 `월간 달력` 제공
- 달력 상단에서 `이전/다음 달`, `이전/다음 연도` 이동 가능
- 날짜 셀에서 일별 점수와 건수 확인 가능
- 날짜 선택 시 `일별 상세 모달` 표시
- 상세 모달에서:
  - 해당 날짜 점수
  - 상승 / 하락 건수
  - 해당 날짜 기록 목록
  - 해당 날짜 전체 기록 초기화

### 3. 통계 탭

- 현재 기간 집계:
  - 이번 주
  - 이번 달
  - 이번 분기
  - 올해
- 최고 기록:
  - 월간 최고 상승 / 최고 하락
  - 분기 최고 상승 / 최고 하락
  - 연간 최고 상승 / 최고 하락
- 통계 카드 크기는 초기안보다 줄이고, 기간 정보는 카드 우상단에 배치

## UX 규칙

### 입력

- 탭: 기록 1건 즉시 추가
- 롱프레스: 즉시 `5건` 입력 후, 누르고 있는 동안 `1초마다 5건` 추가
- 롱프레스 묶음 입력도 `undo snackbar`에서 한 번에 취소

### 되돌리기

- `undo snackbar`는 마지막 추가 행동 1개만 추적
- 단건 입력이면 1건 취소
- 롱프레스 반복 입력이면 해당 배치 전체 취소
- 표시 시간은 약 `5초`

### 정보 제공

- 메인 화면 카피는 최소화
- 상세 기능 설명은 상단 `All We Experience` 이름 옆 `i` 아이콘 모달로 이동
- 메인 입력 관련 아이콘은 가능한 범위에서 엄지 계열 아이콘으로 통일

### 내비게이션

- 하단 탭 3개:
  - `업다운`
  - `달력`
  - `통계`

## 시각 언어

- 브랜드 기본 톤은 `블랙 + 민트/시안`
- 하락 정보는 블루 계열로 분리
- 이 색 문법을 상단 브랜드, 메인 버튼, 탭, 패널 배경에 반영
- 아이콘은 `AWE` 텍스트가 포함된 커스텀 브랜드 이미지 기반
